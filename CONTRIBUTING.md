# Contributing to this project

Thank you for your interest in contributing! Please follow these guidelines:

## Issue Title Guidelines

To keep issues clear and organized, please use the following **title format** when opening a new issue:

### Types

- `bug` → Report a problem or unexpected behavior  
- `feature` → Suggest a new feature or enhancement request  
- `improvement` → Suggestions for code improvements or refactoring 
- `question` → Clarification or help request  

**Tips:**  
- Keep the description short and clear (ideally < 50 characters).   
- Always include the appropriate type prefix.
- Include Python version, OS, and steps to reproduce
